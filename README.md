# GDE
Game Developer Engine written in Go

## Intro
Currently only tested to build on Windows 10 (requires mingw-w64 with gcc) for Windows and Android 4.1 + devices

## Install (WIP)
```
go get gomobile
gomobile init
./build.sh OR build.bat
```
